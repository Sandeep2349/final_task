HTML Glossary

HTML
HyperText Markup Language, the standard markup language for documents designed to be displayed in a web browser.
Element
An individual component of an HTML document, usually consisting of a start tag, content, and an end tag.
Attribute
A modifier of an HTML element that provides additional information about the element, such as "href" or "src".
Semantic HTML
The use of HTML markup to reinforce the semantics, or meaning, of the information in webpages rather than just its presentation.
Void Element
An element that cannot have any child nodes (i.e., no nested elements or text) and does not require a closing tag, such as <br> or <img>.
